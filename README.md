# Projeto-Caixa-Eletrnico
Sistema de Caixa Eletrônico desenvolvido em Java — POO UNICID 2026/1. Simula o funcionamento de um ATM real: saques otimizados por maior cédula, reposição de notas, controle de cota mínima e geração de extrato. Implementa a interface `ICaixaEletronico` integrada a uma GUI fornecida pelo professor.
